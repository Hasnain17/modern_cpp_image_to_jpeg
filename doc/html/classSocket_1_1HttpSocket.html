Detailed Description

template<bool with_https>
class Socket::HttpSocket< with_https >

HttpSocket allows to query a given URL via HTTP (and HTTPS via openSSL) and receive the response.

Encapsulates the C-Socket POSIX API in a RAII fashion. HttpSocket is not movable (buffer array is part of the type) and not copyable (socket fd). For demonstrational purposes, boost_asio ip::tcp::socket is the more sophisticated choice.
